BMW PWA v5 clean icon package

핵심:
- 홈화면 앱 아이콘은 bmw-home-icon-v5-* 새 파일명 사용
- 기존 icon-192.png/icon-512.png/icon-maskable-512.png는 더 이상 참조하지 않음
- manifest도 manifest-v5.webmanifest 새 파일명 사용
- native PWA 자동 스플래시는 bmw-splash-min-v5-512.png로 최소화
- 실제 멋진 인트로는 HTML의 BoMunWildcard 커스텀 인트로만 표시

GitHub에서 권장:
1. 기존 icons 폴더의 오래된 icon-*.png는 삭제해도 됨
2. 이 패키지의 전체 파일 업로드
3. 기존 홈화면 BMW 앱 삭제
4. Chrome/Samsung Internet 사이트 데이터 삭제 권장
5. https://bsk1013.github.io/bmw2026/index.html?v=5 접속 후 홈화면 추가
